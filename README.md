# Etch-A-Sketch 🎨

This project is part of my learning journey through **The Odin Project**.

The goal of this exercise was to practice the fundamentals of **JavaScript DOM manipulation** and **Flexbox layout** by building a simple drawing grid that works like a digital Etch-A-Sketch.

Instead of writing hundreds of div elements manually, the grid is generated dynamically using JavaScript. Each square reacts to mouse movement, allowing the user to "draw" across the grid.

## Features

- 16 × 16 grid generated using JavaScript
- Hover effect that colors the squares
- Adjustable grid size using a button and prompt
- Maximum grid size limit of 100 × 100
- Layout built using Flexbox

## What I Practiced

While building this project, I practiced:

- DOM manipulation
- JavaScript event listeners
- Dynamic element creation
- Flexbox layout
- Handling user input with prompts
- Basic debugging using browser developer tools
